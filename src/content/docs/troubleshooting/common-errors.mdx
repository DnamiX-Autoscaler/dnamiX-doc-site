<DocSection id="prometheus" title="Prometheus Targets Not Found">

<Alert type="error" title="Problem">
Targets do not appear in Prometheus UI.
</Alert>

Possible causes:

- ServiceMonitor labels do not match Service labels
- Wrong namespace
- Metrics endpoint missing

<CodeBlock language="bash" code={`kubectl get svc --show-labels`} />

</DocSection>

<DocSection id="metrics" title="No Metrics Data">

<Alert type="warning" title="Problem">
/metrics endpoint returns empty data
</Alert>

Fix:

- Ensure prom-client is configured
- Check middleware setup

</DocSection>

<DocSection id="istio" title="Istio Metrics Missing">

<Alert type="error" title="Problem">
No service mesh data
</Alert>

Fix:

<CodeBlock language="bash" code={`kubectl label namespace ecommerce istio-injection=enabled`} />

</DocSection>
