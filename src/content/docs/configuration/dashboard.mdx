<DocSection id="overview" title="Overview">

The DnamiX Dashboard provides a real-time interface for DevOps engineers and operators.

It visualizes:

- Cluster resource usage
- Node-level metrics
- Pod-level metrics
- Application-level metrics
- Service mesh traffic
- Graph centrality behavior
- Stress Index
- Dataset export status
- Scaling decisions
- Anomaly scores

</DocSection>
