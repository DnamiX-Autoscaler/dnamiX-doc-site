<DocSection id="overview" title="Overview">

`dnx init` creates the initial DnamiX configuration for your application.

It collects deployment details such as:

- Project name
- Application namespace
- Monitoring namespace
- Services
- Metrics endpoints
- Prometheus configuration
- Istio availability
- Enabled DnamiX components

</DocSection>

<DocSection id="command" title="Command">

<CodeBlock
  language="bash"
  title="Initialize DnamiX"
  code={`dnx init`}
/>

</DocSection>
