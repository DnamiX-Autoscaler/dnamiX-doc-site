<DocSection id="overview" title="Overview">

`dnx deploy` applies the generated DnamiX configuration into the target Kubernetes cluster.

This command deploys:

- Monitoring namespace
- Prometheus stack
- Application ServiceMonitors
- Istio PodMonitors
- PeerAuthentication
- Metrics collector backend
- Monitoring dashboard
- Optional ML prediction, scaling execution, and anomaly detection services

</DocSection>

<DocSection id="command" title="Command">

<CodeBlock
  language="bash"
  title="Deploy DnamiX"
  code={`dnx deploy`}
/>

</DocSection>

<DocSection id="deployment-flow" title="Deployment Flow">

<StepFlow
  steps={[
    {
      title: "Create monitoring namespace",
      description: "Creates a dedicated namespace for Prometheus, Grafana, and DnamiX monitoring components.",
      code: "kubectl apply -f .dinamix/k8s/00-monitoring-ns.yaml"
    },
    {
      title: "Install Prometheus stack",
      description: "Deploys kube-prometheus-stack using Helm values generated by DnamiX.",
      code: "helm install prometheus prometheus-community/kube-prometheus-stack -n monitoring -f .dinamix/k8s/01-kube-prom-stack-values.yaml"
    },
    {
      title: "Apply ServiceMonitors",
      description: "Registers application /metrics endpoints for scraping.",
      code: "kubectl apply -f .dinamix/k8s/02-app-servicemonitors.yaml"
    },
    {
      title: "Apply Istio telemetry monitors",
      description: "Registers Istio and Envoy metrics for service mesh traffic analysis.",
      code: "kubectl apply -f .dinamix/k8s/03-istio-servicemonitors.yaml"
    },
    {
      title: "Deploy DnamiX services",
      description: "Deploys metrics collector, dashboard, prediction service, scaling executor, and anomaly detector.",
      code: "kubectl apply -f .dinamix/components/"
    }
  ]}
/>

</DocSection>

<DocSection id="manual-deploy" title="Manual Deployment Commands">

<CodeBlock
  language="bash"
  title="Manual deployment"
  code={`kubectl apply -f .dinamix/k8s/00-monitoring-ns.yaml

helm install prometheus prometheus-community/kube-prometheus-stack \\
  -n monitoring \\
  -f .dinamix/k8s/01-kube-prom-stack-values.yaml

kubectl apply -f .dinamix/k8s/02-app-servicemonitors.yaml
kubectl apply -f .dinamix/k8s/03-istio-servicemonitors.yaml
kubectl apply -f .dinamix/k8s/05-istio-peer-auth.yaml`}
/>

</DocSection>

<DocSection id="post-deploy" title="After Deployment">

Check if DnamiX components are running:

<CodeBlock
  language="bash"
  code={`kubectl get pods -n monitoring
kubectl get servicemonitor -n monitoring
kubectl get podmonitor -n monitoring
kubectl get svc -n monitoring`}
/>

</DocSection>

<Alert type="warning" title="Do not deploy blindly">
